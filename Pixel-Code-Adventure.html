Level 1: 像素世界

電腦看到的圖片不是一張紙，而是由許多發亮的「小格子」組成的！
這些小格子我們稱之為 Pixel (像素)。

Level 2: 黑白密碼 (RLE)

為了節省空間，電腦會「數」連續有幾個顏色。
這叫做 Run Length Encoding (RLE)！
規則：預設從「白色」開始數。如果是黑色開頭，前面要補 0。

例子：1 白, 3 黑, 1 白

👉 1, 3, 1

Level 3: 彩色編碼

如果是彩色的怎麼辦？我們給顏色編號：
0 = 白色, 1 = 黑色, 2 = 紅色

表示方法：長度 + 顏色代碼
例如：
30 → 3格白色
11 → 1格黑色
42 → 4格紅色

Level 5: 圖檔大小計算機

圖片的檔案大小，取決於「總像素數量」乘上「每個像素的色彩位元數」。
公式：檔案大小 (bit) = 寬度 × 高度 × 色彩位元數
